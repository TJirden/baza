## Команды запуска

**1. Поднятие инфраструктуры**
```bash
docker-compose up -d
```

**2. Загрузка модели эмбеддингов**
```bash
docker exec -it ollama ollama pull all-minilm
```

---

## Сборка и запуск через Gradle

Если инфраструктура в Docker уже запущена, используйте следующие команды для сборки и запуска самого приложения:

**1. Сборка JAR-файла**
```bash
./gradlew clean build
```

**2. Запуск приложения**
```bash
./gradlew bootRun
```

---


## Запуск и переменные окружения (.env)

Компиляция (точка входа )
При компиляции необходимо так же указать `environment variables` любым удобным способом. Список:

* **`ADMIN_PASSWORD`**: Пароль для очистки бд в методе clear (использование `/clear {ADMIN_PASSWORD}`)
* **`TELEGRAM_TOKEN`**: Токен бота
